### คำอธิบายฟีเจอร์
     Registration Report : แสดงผลหน่วยกิตในรายวิชาที่นิสิตได้ลงเรียนตามโครงสร้างหลักสูตร, แสดงผลว่านิสิตยังขาดหน่วยกิตในหมวดใดบ้าง
     Subject Search : นิสิตค้นหารายวิชาต่าง ๆ (รายวิชาที่ขาด) ได้ตามหมวดหมู่ และสามารถกดติดดาวรายวิชาที่สนใจได้
     Activity Transcript : แสดงรายละเอียดกิจกรรมที่นิสิตได้เข้าร่วม พร้อมแสดงชั่วโมงกิจกรรมที่นิสิตได้เข้าร่วมทั้งหมดในหมวดต่าง ๆ ตามโครงสร้างกิจกรรม,แสดงหมวดที่นิสิตยังได้ชั่วโมงกิจกรรมไม่ผ่านเกณฑ์
     KU Event : กดค้นหากิจกรรมตามชื่อ หรือหมวดหมู่ของกิจกรรม, กดเลือกกิจกรรมที่แสดงได้ (กิจกรรมวันนี้ และ กิจกรรมใหม่),  กดติดดาวกิจกรรม
